Práctica 2: Introducción a JavaScript

Introducción

En esta segunda práctica vamos a introducir el uso de JavaScript (JS) para dotar de dinamismo e interacción a nuestra página web. Se pretenden establecer las bases de conocimiento que nos permitirán manejar objetos multimedia en el navegador, así como las interacciones del usuario con ellos.

Se hará uso de los siguientes conceptos estudiados en las clases teóricas:

  • Manipulación del árbol de objetos de la página (DOM)
  • Manejo de eventos

Se pretende desarrollar la página web de una empresa, que hará uso del conocido diseño con pestañas. Es decir, la web constará de varias secciones a las que podremos acceder clickando la pestaña correspondiente. El objetivo de esta práctica es el desarrollo de esta web, tanto su interfaz de usuario como la navegación por la misma mediante pestañas. Para ello, tendremos que hacer uso de un script JavaScript que capture la interacción del usuario con las pestañas y realice las acciones necesarias para visualizar el contenido de esa sección de la página web.

Recomendaciones

  • Durante el desarrollo de esta práctica y las siguientes se recomienda el uso de las herramientas para desarrolladores de Google Chrome, accesibles desde el menú Herramientas, principalmente de la consola JavaScript y el explorador de elementos.

Paso 1: HTML

En primer lugar deberá crearse la estructura HTML correspondiante a la web de la empresa. Esta estructura debe usar los elementos:

  • header para la cabecera de la página
  • nav para las pestañas, que son usadas para la navegación en el documento; para cada opción usaremos un elemento contenedor, por ejemplo span
  • section para las secciones (contenidos) de las distintas pestañas
  • footer para el pie de página

El resultado será una web con todas las secciones, una a continuación de la siguiente. Diseñaremos la navegación por pestañasa partir del siguiente paso.

Paso 2: Visualización

Existen distintas opciones para implementar la interacción de las pestañas, cualquiera de ellas perfectamente válida. Una de estas opciones es el uso de propiedades CSS para mostrar/ocultar contenidos. Para mostrar el contenido de una sección, haremos uso de la propiedad CSS display aplicada sobre el elemento correspondiente a la sección. Esta propiedad tomará el valor block en la sección con contenido a mostrar, y el valor none en las secciones con contenido oculto. Por tanto, crearemos dos estilos CSS para las secciones, uno para mostrar la sección visible y otro para ocultar el resto de secciones.

A continuación, crearemos dos estilos CSS para las pestañas, uno usado para la pestaña activa y otro para el resto. Por ejemplo, puede cambiarse el color de fondo de la misma (background), el borde (border), cambiar el tipo de letra (font), etc.

Finalmente, podemos mejorar la experiencia de usuario mediante el uso de otros recursos CSS:

  • Poniendo las esquinas de las pestañas redondeadas con las propiedades -webkit-border-radius / -moz-border-radius, únicas para cada navegador (si queremos que funcione en varios navegadores deberan usarse todas ellas)
  • Cambiando el comportamiento del cursor al pasar sobre una pestañas (cursor)
  • Aplicando otras mejoras visuales que nos parezcan apropiadas

Paso 3: Eventos

Se desarrollará un script que implemente la nevegación por pestañas en nuestra página. Para ello, haremos uso del elemento script. Para evitar problemas de concurrencia en la ejecución de scripts, se recomienda insertar éstos al final del body.

El script desarrollado deberá asignar al elemento que compone la pestaña (en el ejemplo, el elemento span) un manejador de eventos que se ejecute con el evento click del ratón. Este manejador será una función JavaScript que definiremos a continuación. Recuerda que en el código de esa función podremos acceder al elemento de la pestaña mediante la palabra reservada this. La función que programaremos realizará las siguientes acciones:

  • Resaltará la nueva pestaña activa y desactivará el resto.
  • Ocultará las secciones del documento excepto la seleccionada.

Para ello, podremos hacer uso de la nueva propiedad HTML5 classList que para un elemento devuelve otro elemento con el que podremos manipular su atributo class (estilos). Por ejemplo, var lista = element.classList; devuelve el elemento en la variable lista, sobre la que podremos ejecutar acciones para añadir estilos add, eliminarlos remove o cambiar de estado toggle.

Paso 4: Contenido dinámico

El contenido de una web puede ser dinámico, es decir, que el número de pestañas y su contenido pueden variar en función de algunos parámetros (por ejemplo, de los permisos del usuario de un servicio). Por tanto, esta información puede venir dada desde el servidor de la página, y por tanto va a ser dependiente de una ejecución concreta. Vamos a simular este hecho definiendo dos arrays en el script desarrollado anteriormente: uno que contiene cadenas de texto correspondientes al nombre de las pestañas; y otro del mismo tamaño con el contenido de las mismas (que puede ser código HTML). Debemos, por tanto, construir toda la estrucutra de la página de forma dinámica, ya que tendremos un número variable de pestañas. Un ejemplo de los arrays anteriores es (atención al uso de comillas en cadenas de texto con HTML):

var tabNames = ["Inicio", "Noticias", "Acerca de"];
var tabValues = ["<p>Esta es la página de inicio de la web de nuestra empresa</p>", "<img border='0' src='http://www.w3.org/html/logo/downloads/HTML5_Logo_64.png' alt='HTML5 logo' />", "<p>Este ejemplo ha sido realizado por los profesores de Aplicaciones Multimedia</p>"];

Para implementar esta funcionalidad puede hacerse uso de funciones JavaScript para la manipulación del DOM, como createElement, appendChild, parentNode, etc. o la propiedad innerHTML.

Preguntas de Autoevaluación

Tras la finalización de la práctica, deberíais ser capaces de contestar a estas preguntas:

  • Describe el uso de los siguientes elementos HTML en la página de vuestra empresa:
    • header, nav, section, span, footer
  • Describe el proceso elegido para mostrar/ocultar las pestañas. ¿Puedes pensar en otro procedimiento distinto a usar display? Toma como punto de partida el trabajo realizado en el paso 4.
  • Explica paso a paso cómo has implementado la interacción con las pestañas. ¿Qué eventos has tenido en cuenta? ¿Cómo has asignado un manejador a los eventos? ¿Cuándo se ejecuta el manejador? ¿Cómo accedes los elementos de la pestañas y de su contenido? ¿Cómo accedes/modificas los estilos de un elemento?
  • ¿Qué funciones puedes usar para manejar la estructura del documento? ¿Cómo has creado elementos nuevos en el documento? ¿Cómo los has colocado en la posición que querías? ¿Sabrías asignarles atributos? ¿Y cambiar su estilo?
